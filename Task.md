# 兰州美食文化宣传小程序设计与开发

本题目要求学生使用微信开发者工具，WXML构建前端页面结构，WXSS设置样式，JavaScript用于处理页面逻辑；选择Node.js、Spring  Boot、Mybatis等Java技术栈设计后端，用于处理小程序的后台逻辑和数据库操作。也可以选择使用微信小程序的云开发平台，无需搭建自己的服务器，直接使用微信提供的后台服务；数据库选择使用MySQL或MongoDB。小程序实现的功能包括：用户功能（注册登录、美食分类、热门浏览、美食分享、美食文化、平台公告、美食活动、美食投票、美食团购、特价美食、个人中心）；后台管理（轮播管理、用户管理、美食管理、美食分享管理、美食活动管理、美食预约管理、数据分析）。通过该平台鼓励用户分享与美食相关的故事或记忆，构建美食文化数据库，增强平台的文化内涵；定期举办线上美食文化讲座，邀请专家讲解美食历史、文化背景等，提升用户的美食文化素养；根据用户美食购买记录进行在线个性化的推荐等。
