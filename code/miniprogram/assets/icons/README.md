# 图标资源说明

本目录需要以下图标文件：

- home.png / home-active.png
- category.png / category-active.png
- activity.png / activity-active.png
- profile.png / profile-active.png

## 功能图标 (可选)
- notice.png

- logo.png
- wechat.png
- order.png
- favor.png
- share.png
- vote.png
- setting.png

## 建议
1. 从 iconfont.cn 下载免费图标
2. 或使用阿里巴巴矢量图标库
3. 确保图标为 PNG 格式，透明背景
4. 建议尺寸：64x64 或 128x128 (2x)
